# Personal Account Management

## Description
This project is a Python program for managing a personal bank account using OOP concepts.

## Features
- Deposit and withdraw money
- Track balance
- Store transaction history
- Operator overloading (+ and -)

## Files
- account.py – main classes
- test_account.py – test cases
- uml_diagram.png – UML class diagram

## How to Run
```bash
python account.py
